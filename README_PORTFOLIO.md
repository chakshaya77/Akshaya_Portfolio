# Akshaya Chittimilla - Portfolio

A premium, cinematic, full-stack developer portfolio.

## Features
- **Cinematic Experience:** Built with Three.js for interactive particle backgrounds and smooth page transitions.
- **Glassmorphism UI:** Premium aesthetic utilizing backdrop blurs and subtle white glows.
- **Dynamic Projects:** Projects are fetched dynamically from a secure MongoDB database.
- **Secret Admin Gateway:** A hidden entry point located on the Navbar emblem provides access to a secure Admin Dashboard.

## Admin Features
- **Authentication:** Secure login using JWT and Admin credentials.
- **Upload Projects:** Add new projects to the portfolio dynamically with a name, description, live link, and cover image.
- **Manage Projects:** View and delete existing projects from the portfolio.

## Tech Stack
- **Frontend:** React, Vite, React Router, CSS, Three.js, Lucide-React
- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT
